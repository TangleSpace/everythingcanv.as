Instructions

  • [LMB] to draw
  • [ALT + LMB] to rotate
  • [Scroll / ALT + CTRL + LMB] to zoom in/out
  • [CTRL + LMB] to pan up and down
  • [CTRL Z / CTRL Y] undo and redo
  • [SHIFT] toggle stroke select
  • [SPACE] to toggle UI
  • Tablets and mobile supported
  • gltf exports DO NOT retain shader effects
  • gltf exports DO retain animation data, but they don't import properly into blender (other apps untested)
  • download and run this python script to attach the animations to the objects.
  • drag and drop a glb or gltf on the scene to use a custom draw object
  • drag and drop an img to make the draw object an image plane
  • drag and drop a goe.ink txt to reload a saved project
  • most models by David Oreilly's Everything Library
  • site by Vince Mckelvie
